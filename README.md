
# Securing Your Home Network

Securing Your Home Network is an education app designed for adults to learn about home internet networks and how to best protect themselves. This project was created as a part of the CPRE 5300 course at Iowa State University. 

A video of the app working and the up-to-date APK can be found in the submission directory. 


## Author

- [@Garrett Thompson](https://www.github.com/garrett-lane)
    Cyber Security Engineering undergraduate student at Iowa State University


## Installation

Install this app on an Android device by downloading the APK from the submission directory and executing the file locally. Although the APK is signed, Android will likely warn of potential dangers. This app is not harmful and requires no permissions, however install at your own risk.
